Trinity AI

Decentralized AI Chat Platform

Trinity is a fully decentralized AI chat application built on the Internet Computer (ICP). It pairs a React 19 frontend with a Rust-based backend canister, delivering a ChatGPT-style conversational interface that runs entirely on-chain with no centralized servers.

Users interact with AI models through a clean, responsive UI while their session data lives on the decentralized web. The app supports real-time streaming responses, dark mode, and persistent chat history stored in canister memory.

Retro Arcade

Browser-Based Game Emulation Platform

A full-featured retro gaming arcade that lets you play N64, NDS, and PSP games directly in the browser. ROMs are served via signed URLs from Supabase Storage and Cloudflare R2, while EmulatorJS handles emulation with WASM-compiled RetroArch cores.

Features include cloud save sync (auto-uploads save files to the server every 15 seconds), online multiplayer via WebRTC netplay, Bluetooth controller support with per-game button remapping, and a guest mode for quick demos without an account.

2D Triangle Shooter

Canvas-Based Arcade Game

A lightweight browser game built with the HTML5 Canvas API. You control a green triangle that rotates and fires projectiles at bouncing dots. Each hit makes your triangle grow larger. Movement uses WASD, rotation uses arrow keys, and spacebar shoots.

Built entirely in vanilla JavaScript with no dependencies — just raw canvas drawing, collision detection, and a tight game loop running on requestAnimationFrame.

ThreeJS Demo

First-Person 3D Environment

A first-person 3D corridor built with Three.js. Walk through a walled hallway, dodge blue obstacles that slide toward you, and jump over them with physics-based arc movement. Collision detection keeps you inside the world boundaries and prevents you from walking through walls.

Demonstrates 3D scene composition, perspective camera controls, real-time collision with bounding boxes, and procedurally spawned obstacles — all running in WebGL with no build step required.
